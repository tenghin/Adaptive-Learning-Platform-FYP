FYP AI POWERED PERSONALISED LEARNING PLATFORM

This file documented the planning for the FYP project

<------------| AI Feature |------------>
    Generating Visual Learning material aka Mind map

        AI Mind Map Pipeline

            Lesson Content
                ↓
            Prompt Builder
                ↓
            Gemini API
                ↓
            Knowledge JSON
                ↓
            Store in PostgreSQL
                ↓
            Load JSON
                ↓
            Convert to React Flow Nodes + Edges
                ↓
            Dagre (Auto Layout)
                ↓
            React Flow
                ↓
            Interactive Mind Map

        Responsibilities:

            --------Gemini--------
            Understand lesson
            Generate structured JSON
            No UI or coordinates

            ---------JSON----------
            Store concepts
            Store relationships
            Reusable for quiz, recommendations, search, etc.

            -------Converter-------
            JSON → React Flow nodes & edges

            ---------Dagre---------
            Automatically generate x & y positions    #react flow requires an X and Y coordinates for each item, this is to create those Automatically
            No manual positioning

            -------React Flow-------
            Render graph
            Drag
            Zoom
            Click nodes
            Future expand/collapse

            note: mermaid isnt being used due to it lack proper visualisation and lack Interactivity. It also have consistency issue.

    Generate and receive prompt from selected AI
        
        AI Architecture

            Frontend
                ↓
            Flask Route
                ↓
            AI Service
                ↓
            Prompt Builder
                ↓
            Gemini API
                ↓
            JSON Response
                ↓
            Backend Validation
                ↓
            Frontend
        
        Folder Layout
            adaptive_learning/

            ai/
            │
            ├── ai_service.py
            ├── prompt_builder.py
            ├── schemas.py
            └── gemini_client.py

        Responsibilities
        
            gemini_client.py
                Connect to Gemini API
                Send prompt
                Return AI response
            
            prompt_builder.py

                Store all prompts.

                Examples:

                build_graph_prompt()
                build_summary_prompt()
                build_quiz_prompt()
                build_explanation_prompt()
            
            ai_service.py

                AI workflow.

                Lesson
                    ↓
                Build Prompt
                    ↓
                Call Gemini
                    ↓
                Validate JSON
                    ↓
                Save Database

            schemas.py

                Store expected JSON structures.

                Examples:

                Knowledge Graph
                Summary
                Quiz
                Prompt Format
                Role
                ↓

                Task
                ↓

                Rules

                - Return JSON only
                - No Markdown
                - Follow schema exactly

                ↓

                Lesson Content
